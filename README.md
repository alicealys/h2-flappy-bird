# h2-flappy-bird

Based on [Black Ops III Flappy Bird by JariK](https://steamcommunity.com/sharedfiles/filedetails/?id=1720319747)

# Keybinds

* **SPACE** or **MOUSE1**: Jump
* **MOUSE2**: Change bird skin (only during pre game)
